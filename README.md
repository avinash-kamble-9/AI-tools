Here are the **AI tools with their short use cases**

| AI Tool           | Short Use                                                                                                                         |
| ----------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| **Claude Code**   | Understands entire codebases and systems; helps design pipelines, debug logs, and generate boilerplate data engineering projects. |
| **DeepSource**    | AI-powered static analysis for **code quality, test coverage, and bug detection** in Python/SQL pipelines.                        |
| **Soda GPT**      | Detects **data quality issues** and analyzes anomalies in datasets automatically.                                                 |
| **Monte Carlo**   | Uses ML to monitor **data pipelines**, detect schema changes, volume drops, and freshness issues.                                 |
| **Notion AI**     | Automatically generates **technical documentation** for pipelines and projects.                                                   |
| **Whimsical**     | Creates **architecture diagrams, flowcharts, and data models** from text prompts.                                                 |
| **dbdiagram**     | Generates **ER diagrams and database models** from SQL or schema definitions.                                                     |
| **Perplexity AI** | Research tool for **tech stack comparison, architecture decisions, and documentation search**.                                    |
| **Runway**        | Generates **automated summaries and insights** from pipeline health data.                                                         |
| **ChatPDF**       | Ask questions from **large technical PDFs and documentation instantly**.                                                          |

✅ **Simple idea:**
These tools help data engineers in **coding, debugging, data quality, architecture design, documentation, and research** to become more productive.
